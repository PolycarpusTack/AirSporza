Key Interactions

Click empty area → Create slot at that time/channel
Click slot → Select, show editor panel
Drag slot body → Move (vertical=time, horizontal=channel)
Drag bottom edge → Resize duration
Right-click slot → Context menu (Edit, Delete, Duplicate)
Ctrl+Z / Ctrl+Y → Undo / Redo via draft operations
5-min snap → All drag/resize snaps to 5-minute grid
